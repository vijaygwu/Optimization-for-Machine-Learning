\section*{Three Reading Paths}

\textbf{{\large\textcolor{primaryblue}{\faGraduationCap}}\ The Theory-First Path} (Chapters 1--3, then 4--7)
\begin{itemize}[itemsep=0.3em]
    \item Start with convex optimization theory for solid foundations
    \item Progress through gradient methods before adaptive optimizers
    \item Estimated time: 4--6 weeks at 4 hours/week
    \item Best for: Building deep understanding of optimization
\end{itemize}

\vspace{0.4cm}

\textbf{{\large\textcolor{primaryblue}{\faCode}}\ The Practitioner Path} (Chapters 2, 4, 5, then others as needed)
\begin{itemize}[itemsep=0.3em]
    \item Start with SGD and Adam---the algorithms you use daily
    \item Add regularization techniques, then fill in theory gaps
    \item Best for: Working engineers who need practical knowledge fast
\end{itemize}

\vspace{0.4cm}

\textbf{{\large\textcolor{primaryblue}{\faBook}}\ The Deep Dive Path} (Focus on one topic at a time)
\begin{itemize}[itemsep=0.3em]
    \item Use this book as a reference for specific optimization challenges
    \item Each chapter is self-contained with cross-references as needed
    \item Best for: Researchers tackling specific optimization problems
\end{itemize}

\section*{Prerequisites Check}

Before diving in, ensure you're comfortable with:
\begin{itemize}[itemsep=0.3em]
    \item Computing gradients of simple functions
    \item Basic matrix operations (multiplication, transpose, inverse)
    \item The definition of convex functions
    \item Expected values and variance
\end{itemize}

If any of these feel unfamiliar, review the Math Recap chapter or return to Book 1: \textit{The Math That Powers AI}.

\section*{Special Features}

Throughout this book, you will encounter these learning aids:

\begin{itemize}[itemsep=0.5em]
    \item {\large\textcolor{thmframe}{\faBookOpen}}\ \textbf{Theorems \& Definitions} --- Formal statements with rigorous proofs
    \item {\large\textcolor{exframe}{\faLightbulb}}\ \textbf{Intuition Boxes} --- Build geometric and conceptual understanding
    \item {\large\textcolor{exframe}{\faPencilAlt}}\ \textbf{Worked Examples} --- Step-by-step calculations you can follow
    \item {\large\textcolor{primaryblue}{\faPlay}}\ \textbf{Algorithm Traces} --- Watch optimizers execute step by step
    \item {\large\textcolor{warnframe}{\faExclamationTriangle}}\ \textbf{Common Mistakes} --- Pitfalls that derail training
    \item {\large\textcolor{primarygreen}{\faQuestionCircle}}\ \textbf{Why Should I Care?} --- Connect theory to practice
    \item {\large\textcolor{accentorange}{\faLaptopCode}}\ \textbf{Try It Yourself} --- Hands-on coding exercises
\end{itemize}
